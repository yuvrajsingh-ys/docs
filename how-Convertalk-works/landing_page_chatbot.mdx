1. To create your Landing page chatbot and manage all your chatbots, you first need to visit the  Convertalk Dashboard

![6 Landing Page Chatbot Convertalk Resource Centre 1 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-1.png)

2. To start creating your new chatbot, click **"Create New Bot."**
3. Select the primary purpose for creating your bot and proceed further.

![6 Landing Page Chatbot Convertalk Resource Centre 2 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-2.png)

4. Select the platform: **“Website Chatbot”**

![6 Landing Page Chatbot Convertalk Resource Centre 3 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-3.png)

5. You will see a notification: **"Bot has been successfully created.”**
6. Select the website chatbot you have created from sidebar.

![6 Landing Page Chatbot Convertalk Resource Centre 4 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-4.png)

7. Click on **“Install Your Chatbot”.**

![6 Landing Page Chatbot  Convertalk Resource Centre 5 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-5.png)

8. Select the **“Landing Page Bot”.**

![6 Landing Page Chatbot  Convertalk Resource Centre 6 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-6.png)

**Test:** To see how your chatbot will function and appear on your Landing page, click on the **"Test"** button.

![6 Landing Page Chatbot  Convertalk Resource Centre 7 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-7.png)

9. Copy URL: Click on **“Copy URL”** and you can share this URL with your users.

![6 Landing Page Chatbot  Convertalk Resource Centre 8 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-8.png)

#### **You have created your Landing Page Chatbot.**

---

### **Embedded Chatbot**

1. Now, if you want to add a chatbot to your landing page, click on **“Embedded Chatbot”**

![6 Landing Page Chatbot  Convertalk Resource Centre 9 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-9.png)

- **Get Chatbot Installation code:** From this section, you can directly copy the script code for your chatbot.
- **Share Installation code: I**nstallation code will be sent directly to your development team via email.
- **Ask  Convertalk Support:** Our team will assist you in installing your chatbot.

Copy the provided script code and find the HTML file such as the header and paste the code that you copied inside the body tag.

![6 Landing Page Chatbot  Convertalk Resource Centre 10 Pn](/images/how-botpenguin-works/create-newchatbot/6.landing_page_chatbot__Convertalk_resource_centre-10.png)

Paste the code inside the body tag, then save and deploy your updated code.

Congratulations\! You have deployed your landing page\!
