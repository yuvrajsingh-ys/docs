### **Steps to Integrate GitBook with  Convertalk**

#### **Step 1: Navigate to Your  Convertalk Account**

1. Log in to your  Convertalk account.
2. \
   Select the chatbot you wish to integrate with GitBook from your list of bots.

#### **Step 2: Access Third-Party App Integrations**

1. In your selected chatbot’s settings, navigate to the **Third-Party App Integrations** section.
2. \
   Use the search bar to find **GitBook**.
3. \
   Click on the **Connect** button next to GitBook.

#### **Step 3: Generate an API Access Token in GitBook**

To integrate GitBook, you’ll need an API access token. Follow these steps:

1. Log in to your GitBook account.
2. Go to the **Settings** section of your GitBook account.
3. \
   Navigate to the **Developer Settings** section.
4. \
   Click on the **Create New Token** button.
   - Name the token appropriately to identify its purpose.
   - Set the required permissions for the token.
5. \
   Copy the generated API access token.

#### **Step 4: Connect GitBook with  Convertalk**

1. Return to the  Convertalk dashboard.
2. Paste the copied API access token into the field provided in the GitBook integration setup.
3. Click on the **Connect** button to finalize the integration.

#### **Confirmation**

Once the integration is successful, you’ll receive a confirmation message. Your chatbot can now access the data and content from your GitBook to improve its training and responses.

### **Troubleshooting**

If you encounter any issues during the integration:

1. Ensure that the API token has the necessary permissions in GitBook.
2. Double-check that the API token is correctly copied and pasted.
3. Verify that you have an active internet connection and are logged into both  Convertalk and GitBook.
