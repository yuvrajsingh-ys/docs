**Integration of  Convertalk with First Promoter**

By completing this integration, all leads generated through your  Convertalk chatbot will automatically be listed in the "Promoters" section of First Promoter.

### **Steps to Integrate First Promoter with  Convertalk**

**Step 1: Navigate to the  Convertalk Dashboard**

1. Log in to your  Convertalk account.
2. \
   On the dashboard, select the chatbot you want to integrate with First Promoter.

**Step 2: Access Third-Party Integrations**

1. Go to the **Third-Party Apps Integration** section in the  Convertalk dashboard.
2. \
   Use the search bar to look for "First Promoter."
3. \
   Click on the **Connect** button next to First Promoter.
4. \
   Now here you have to fill in the API key for the FirstPromoter platform.

**Step 3: Obtain the First Promoter API Key**

1. Log in to your First Promoter account.
2. Navigate to the dashboard and click on your profile icon in the top-right corner.
3. \
   Select **Settings** from the dropdown menu.
4. \
   Go to the **Integrations** section within the settings page.
5. \
   Locate your API key in this section.
6. Copy the API key to your clipboard.

**Step 4: Connect First Promoter to  Convertalk**

1. Return to the  Convertalk dashboard.
2. Paste the copied API key into the API key field provided in the First Promoter integration setup.
3. Click the **Connect** button.

**Step 5: Add Parameters and Save the Integration**

1. Once connected, youâ€™ll be prompted to add questions or parameters related to your lead generation process.
2. \
   After filling in the necessary details, click on the **Save** button.

**Step 6: Test the Integration**

1. Use the **Test Lead** button to ensure the integration is working correctly.
2. \
   A demo lead will be created in your First Promoter account for verification.

#### **Conclusion**

Following these steps will successfully integrate First Promoter with your  Convertalk chatbot. Once integrated, all leads captured by the chatbot will automatically appear in your First Promoter account under the "Promoters" section.
